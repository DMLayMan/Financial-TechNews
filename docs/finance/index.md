## 高收益候选（每日筛选，0-3 只）

目标仓位：**10-20%**（在核心持仓之外的进攻性仓位）。单只建议不超过 10%。

四重硬门槛，缺一即淘汰：

1. **明确催化剂** — 近 30 天内具体基本面事件（财报、产品、订单、政策、并购等）
2. **技术面确认** — 成交放量 + 突破/强势结构
3. **风险可控** — 明确止损位，下行空间 ≤ 15%
4. **信息可查证** — 每条理由都附可访问的来源链接

!!! warning "诚实原则"

    **如果当天没有同时满足四条门槛的候选，报告会直接写"今日无合格候选"。** 宁可空仓等待，也不会为了凑数降低标准、编造数据或给出空洞的"长期看好"式结论。

## 报告结构

每篇报告固定包含六个章节：

1. **执行摘要** — 昨日模型准确率 + 今日大方向 + 重点关注标的 + 候选摘要
2. **昨日复盘表** — 核心 13 只 + 昨日高收益候选的实盘对比
3. **今日核心持仓预测** — 13 只每只：方向、催化剂、操作建议、关键位
4. **今日高收益候选** — 0-3 只，含入场区间/止损位/目标位/仓位/风险点；或"今日无合格候选"
5. **宏观提示** — 利率、美元、期货、行业头条
6. **风险提示** — 免责声明
