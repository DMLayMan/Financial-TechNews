# Financial & Tech News Archive

> 每日由 Claude Cowork 定时生成的两类报告归档站点。

## :material-chart-line: [财经归档](finance/index.md)

每个交易日北京时间 **08:30** 运行。覆盖 13 只港美股核心标的 + 每日筛选高收益候选：

- **美股核心 (7)**：NVDA · MSFT · GOOGL · META · AAPL · AMZN · TSLA
- **港股核心 (6)**：腾讯 (0700) · 阿里 (9988) · 美团 (3690) · 小米 (1810) · 比亚迪 (1211) · 智谱 (02513)
- **高收益候选 (0-3)**：目标总仓位 10-20%，严格四重筛选（基本面催化 + 技术面确认 + 明确止损 + 可查证）。**无合格候选时会诚实写"今日无合格候选"，不会凑数。**

每篇报告包含：昨日模型建议复盘 → 今日核心持仓方向判断 → 高收益候选 → 宏观提示。

## :material-robot-outline: [AI 科技归档](ai-tech/index.md)

每天北京时间 **09:00** 运行。精选过去 24 小时内 **1-2 件最核心的 AI 事件**，做双视角拆解：

- **产品视角**：用户价值、核心场景、商业模式影响
- **工程视角**：模型架构、基础设施、实现细节
- **可深挖方向**：3-5 条带资源指针的子问题清单
- **一句话判断**：So What？真趋势还是炒作？
